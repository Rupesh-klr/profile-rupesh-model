# 📚 Profilo Designer — Documentation

**Profilo Designer** is a reusable, JSON-driven, bilingual (English / తెలుగు) React profile
website template. No backend — it builds to plain static files you can host anywhere.

This `docs/` folder is the full guide. Read it in order:

| # | Document | What's inside |
|---|----------|---------------|
| 1 | [Project Overview](01-project-overview.md) | What this is, the tech, the folder layout |
| 2 | [Getting Started](02-getting-started.md) | Install + run the project on your machine |
| 3 | [Customization Guide](03-customization-guide.md) | Edit content, photos, colours, languages, sections |
| 4 | [Deploy to Hostinger](04-deployment-hostinger.md) | The one-click ZIP, `.htaccess`, upload steps |
| 5 | [First Client — Vedika Raksha](05-first-client-vedika-raksha.md) | The first profile built on this template |
| 6 | [Client Intake Questionnaire](06-client-intake-questionnaire.md) | Bilingual questions to collect all profile details from the client |

## Quick links

- **Run locally:** `npm install` then `npm run dev`
- **Package for hosting:** double-click `Build-Deploy-Zip.bat`
- **Put it on GitHub:** double-click `Init-Git-And-Push.bat`
- **All content lives in:** [`src/data/profile.json`](../src/data/profile.json)
